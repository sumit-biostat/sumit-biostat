# SAS-Clinical-Biostatistics-TLFs

Project: SAS Clinical Biostatistics TLFs

Objective:
Perform statistical analysis on lung cancer dataset and generate TLFs.

Analysis Performed:
- Descriptive statistics
- Frequency tables
- Chi-square tests
- T-tests and ANOVA
- Regression and logistic regression
- Risk score modeling

Outputs:
- Tables (RTF)
- Listings
- Figures

Tools:
- Base SAS
- SAS STAT
